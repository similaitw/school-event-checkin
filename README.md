🏆 學校運動會 / 活動貴賓即時報到系統

這是一個基於 Google Apps Script (GAS) 與 Google Sheets 構建的無伺服器 (Serverless) 即時報到系統。專為學校運動會、頒獎典禮或各類活動設計，讓接待處與司儀台能夠即時同步貴賓資訊。

無需架設主機、資料庫，完全免費且開源，只要有 Google 帳號即可部署使用。

✨ 主要功能

📋 接待處中控台 (Admin)

即時報到：輸入姓名與頭銜即可快速報到，支援活動切換。

智慧輸入：內建常見頭銜選單（校長、會長、議員...），並支援歷史活動名稱快選。

資料管理：

支援修改姓名、頭銜與活動名稱。

軟刪除機制：刪除資料會移至「回收桶」，可隨時救回，防止誤刪。

狀態切換：一鍵標記貴賓「在場」或「已離席」。

資料匯出：

匯出 CSV：供 Excel 整理使用。

匯出 HTML 報表：生成精美的排版名單，方便列印或存檔。

即時同步：採用「樂觀更新 (Optimistic UI)」技術，操作流暢不卡頓。

🎤 司儀儀表板 (Dashboard)

即時顯示：大螢幕優化介面，即時顯示最新報到的貴賓。

視覺化標示：頭銜與姓名顏色區分，重要資訊一目瞭然。

司儀控場工具：

狀態標記：可標記「⭐ VIP」、「✅ 已介紹」、「🎤 已致詞」，狀態即時同步。

複選篩選：可同時篩選「VIP」且「尚未介紹」的名單，避免漏唸重要貴賓。

自動更新：背景自動同步最新資料，無需手動重新整理。

🛠️ 技術架構

前端：HTML5, CSS3, Vanilla JavaScript (無須編譯，直接開啟即可使用)。

後端：Google Apps Script (GAS)。

資料庫：Google Sheets (Google 試算表)。

🚀 部署教學 (Installation)

步驟 1：準備 Google Sheet

建立一個新的 Google Sheet。

將下方的工作表名稱改為 報到名單。

設定第一列 (Row 1) 標題，請務必依照以下順序：

A1: ID

B1: 時間

C1: 姓名

D1: 頭銜

E1: 狀態

F1: 活動名稱

G1: 標記

步驟 2：設定後端程式 (GAS)

在試算表中，點擊選單 「擴充功能」 > 「Apps Script」。

將專案中的 Code.gs 程式碼複製並貼上，覆蓋原有內容。

點擊右上角 「部署」 (Deploy) > 「新增部署」 (New deployment)。

設定如下：

類型：網頁應用程式 (Web app)。

執行身分：我 (Me)。

誰可以存取：任何人 (Anyone) (這點非常重要，否則網頁無法連線)。

點擊部署，並授權存取。

複製產生的 網頁應用程式網址 (Web App URL)。

步驟 3：設定前端網頁

下載本專案的 admin.html 與 dashboard.html。

使用文字編輯器 (如 Notepad, VS Code) 開啟這兩個檔案。

找到程式碼最上方的變數設定：

const SCRIPT_URL = '請將你的_GOOGLE_APPS_SCRIPT_網址貼在這裡';


將引號內的文字替換為步驟 2 複製的網址。

儲存檔案。

📖 使用說明

1. 接待人員

使用瀏覽器開啟 admin.html。

輸入或選擇「當前活動名稱」。

輸入貴賓頭銜與姓名，點擊「確認報到」。

若需修改或刪除，請在下方列表操作。

2. 司儀 / 主持人

使用平板或筆電開啟 dashboard.html。

選擇對應的活動名稱。

利用上方的勾選框篩選名單 (例如：勾選「⭐ 僅顯示 VIP」與「✅ 尚未介紹」)。

介紹完畢後，點擊卡片上的「✅ 已介紹」按鈕，狀態會自動更新。

請勿隨意更動 Google Sheet 的欄位順序，以免程式讀寫錯誤。

若修改了 Code.gs 後端程式碼，務必重新部署 (管理部署 > 編輯 > 建立新版本)，舊網址才會生效。
